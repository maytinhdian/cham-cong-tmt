# Naming Conventions

## Classes

PascalCase

Examples:

Employee
AttendanceLog
ShiftMatcher

## Services

Suffix:

Service

Examples:

EmployeeService
AttendanceLogService

## Actions

Suffix:

Action

Examples:

CreateEmployeeAction
AssignShiftAction

## DTO

Suffix:

Data

Examples:

EmployeeData
AttendanceResultData

## Database

Tables:

snake_case plural

Examples:

employees
attendance_logs
attendance_results

Columns:

snake_case

Examples:

employee_id
check_in_time
ot_minutes
